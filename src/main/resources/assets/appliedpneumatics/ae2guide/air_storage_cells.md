# 空气存储元件

<Row>
    <ItemImage id="air_cell_1k" scale="4" />
</Row>

空气存储元件为ME网络提供空气存储能力，每字节可存储250ml空气。当空气存满时继续向元件中存入空气，会导致元件
从驱动器中弹出，甚至会导致驱动器爆炸。

## 便携存储元件
便携存储元件可以插入驱动器当做正常的空气存储元件，也能当作最高压强为20bar的气罐，直接与气动工艺的机器进行
交互。

## 可用升级卡
- [安全卡](upgrades.md)：保护你的驱动器和存储元件，当空气存满后自动拒绝更多存入；
- [真空卡](upgrades.md)：当空气存满后仍向其中存入空气，则会自动销毁多余的部分。可以和安全卡一样保护你的驱动器；
- [充气卡](upgrades.md)：用于便携存储元件，消耗能量和其中的空气，为玩家背包中消耗气体的物品充气。
