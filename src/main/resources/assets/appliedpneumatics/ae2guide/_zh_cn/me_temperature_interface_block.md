# ME温控接口

<Row>
  <BlockImage id="me_temperature_interface" scale="4" />
</Row>

ME 温控接口会通过消耗 **ME 网络** 中存储的 **空气和能量**，  
将自身的温度调节到设定的 **期望值**。

每改变 **1 J 热量**，大约需要消耗 **25 AE 能量** 和 **19 mL 空气**。

## 可用升级卡
- **加速卡**：提升每 tick 的最大温度改变量。
- **容积卡**：提升接口的基础热容。
