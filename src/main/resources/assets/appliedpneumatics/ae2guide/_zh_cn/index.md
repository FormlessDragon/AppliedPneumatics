# 应用气动

  <BlockImage id="me_amadron_process_station" scale="4" />
  <BlockImage id="me_pressure_interface_block" scale="4" />
  <BlockImage id="me_temperature_interface" scale="4" />

这个附属提供了 **应用能源2 (AE2)** 和 **气动工艺：再压缩 (PNC:R)** 的桥接。
它能让你使用ME网络控制 **压缩空气** 、 **温度** 以及 **亚马龙** 系统。

## 内容速览

- **空气存储元件** (1k~256m) — 将压缩空气直接存储到ME网络中。
- **便携空气存储元件** — 随身的超大容量气罐，最大压为20bar。
- **ME气压接口** — 通过与ME网络交互，将气压维持在你的设定值。
- **ME温控接口** — 消耗压缩空气和能量，将温度维持在你的设定值。
- **P2P通道** — 通过AE的P2P通道传输温度与压缩空气。
- **亚马龙处理站** — 将亚马龙系统接入到AE的自动合成网络中。
- **新增联动配方** — 一系列新增配方，包含亚马龙交易和装配室配方。
