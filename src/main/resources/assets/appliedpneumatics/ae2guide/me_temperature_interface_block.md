# ME温控接口

<Row>
  <BlockImage id="me_temperature_interface" scale="4" />
</Row>

ME温控接口通过消耗ME网络中存储的空气和能量将自身温度调节到期望值。每改变1J热量，其会消耗约25AE能量以及19毫
升空气。

## 可用升级卡
- 加速卡：提高每tick最大温度改变量；
- 容积卡：自高自身基础热容。
